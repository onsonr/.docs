#### P

<Accordions>
<Accordion title="Public Key Assertion (WebAuthn)">
**PublicKeyAssertion** in WebAuthn is cryptographic proof of private key possession to verify identity without passwords.
</Accordion>

<Accordion title="Public Key Attestation (WebAuthn)">
PublicKeyAttestation in WebAuthn verifies a credential's authenticity via an attestation statement from the authenticator.
</Accordion>
</Accordions>

#### R

<Accordions>
  <Accordion title="Relaying Party">
    An entity that relies upon a verifiable credential or authenticated identity
    during a transaction.
  </Accordion>
</Accordions>

#### W

<Accordions>
  <Accordion title="WebAuthn">
    A web standard for authenticating users using public-key cryptography
    instead of passwords.
  </Accordion>
</Accordions>
